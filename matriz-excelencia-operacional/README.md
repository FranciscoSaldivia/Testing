# Matriz de Excelencia Operacional

Aplicación web (React + Vite + TypeScript + Tailwind) para visualizar la matriz de excelencia operacional.

## Requisitos

- [Node.js](https://nodejs.org/) v18 o superior (incluye `npm`)

## Instalación y ejecución local

```bash
npm install
npm run dev
```

Esto levanta un servidor de desarrollo (por defecto en `http://localhost:5173`).

## Otros comandos

```bash
npm run build     # genera la versión de producción en /dist
npm run preview   # sirve localmente el build de producción
```
